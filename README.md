# 💰 Expense Tracker Web App

A simple and modern expense tracker built using Flask.

---

## 🚀 Features
- Add expenses
- View all expenses
- Monthly summary
- Category-wise pie chart
- Responsive UI

---

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS
- Chart.js

---

## ▶️ How to Run

1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
